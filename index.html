Yoda Metadata Editor

This is the Yoda metadata editor created by Utrecht University and is open to anyone (no affiliation required). If you are already using Yoda, please use the Yoda metadata editor from within Yoda. You can use this editor to create metadata for your data package and download it to a JSON file, which you can then add to your data package. Note that nothing is stores, saved, or sent anywhere while filling in or downloading this metadata. If you close this page before downloading, you will loose the content already filled in. If you want to edit a previously downloaded Yoda metadata JSON, please use the upload button (check that all fields are correctly entered), edit the required fields, and download to JSON again. The source code for this Yoda metadata editor can be found at here.
